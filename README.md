# Active Directory Lab

## 📌 Project Overview

This project simulates a real enterprise network using virtual machines.

## 🧱 Lab Architecture

* Domain Controller: Windows Server 2025
* Client Machine: Windows 10
* Network: Host-only (internal network)

## ⚙️ Technologies Used

* VMware Workstation
* Windows Server 2025
* Windows 10
